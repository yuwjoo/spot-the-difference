# 🐑 羊了个羊游戏

这是一个基于Vue 3和TypeScript开发的羊了个羊游戏实现，使用项目中已有的贴纸资源作为游戏元素。

## 🎮 游戏简介

羊了个羊是一款经典的三消类游戏，玩家需要在限定时间内将棋盘上的所有元素消除。游戏规则简单但具有挑战性，需要玩家合理规划消除顺序。

## ⚙️ 核心功能

### 1. 游戏玩法
- 点击棋盘上的图案，将其放入下方选中区域
- 当选中区域中有3个相同图案时，这些图案会自动消除
- 在规定时间内消除所有图案即可通关
- 选中区域最多可容纳7个图案
- 时间用完或选中区域满且无法消除时游戏结束

### 2. 游戏特性
- **关卡系统**：支持多关卡游戏，每关难度递增
- **计时挑战**：每关有时间限制，增加游戏紧张感
- **洗牌功能**：可以打乱棋盘上的元素位置
- **提示系统**：提供有限次数的提示，帮助玩家找到可消除的元素
- **分数计算**：根据消除速度和消除组数计算分数
- **响应式设计**：适配不同屏幕尺寸，支持移动端和桌面端
- **动画效果**：包含匹配动画、悬停效果等增强用户体验
- **帮助说明**：提供游戏规则和操作说明

### 3. 技术实现
- 使用Vue 3 Composition API进行状态管理
- TypeScript提供类型安全
- 组件化设计，提高代码可维护性
- 响应式样式，确保在不同设备上的良好显示
- 动画和过渡效果增强用户体验

## 🎯 游戏规则

1. **选择元素**：点击棋盘上可见的元素将其放入选中区域
2. **消除规则**：当选中区域中有3个相同类型的元素时，它们会自动消除
3. **游戏目标**：在规定时间内消除棋盘上的所有元素
4. **失败条件**：
   - 时间用尽
   - 选中区域填满（7个位置）且无法消除任何元素
5. **道具使用**：
   - 洗牌：打乱棋盘上元素的位置
   - 提示：高亮显示一个可选择的元素

## 🚀 如何运行

游戏已经集成到spot-the-difference项目中，可以通过以下步骤运行：

1. 确保项目依赖已安装：`npm install`
2. 启动开发服务器：`npm run dev`
3. 访问游戏页面：在项目路由中导航到羊了个羊游戏页面

## 🎨 游戏界面

游戏界面包含以下主要部分：
- **游戏头部**：显示关卡、分数和剩余时间
- **游戏棋盘**：包含所有可交互的游戏元素
- **选中区域**：显示当前选中的游戏元素（最多7个）
- **控制按钮**：包含洗牌、提示和重新开始按钮
- **帮助弹窗**：显示游戏规则说明
- **游戏结束弹窗**：显示游戏结果和相关统计信息

## 🔧 游戏配置

游戏的主要配置参数包括：
- 棋盘尺寸和元素大小
- 最大选中元素数量（7个）
- 最小匹配数量（3个）
- 每关时间限制（随关卡递增而减少）
- 元素旋转角度范围
- 提示次数和冷却时间

## 📦 资源使用

游戏使用项目中`src/utils/sticker`目录下的贴纸资源作为游戏元素，包括：
- 原始图片（originalImage）
- 掩码图片（maskImages）
- 修改后的图片（modifiedImages）

这些资源通过`useStickers()`函数获取并在游戏中使用。

## 🎮 玩家技巧

1. 优先消除底层元素，避免上层元素被完全覆盖
2. 保持选中区域有足够的空位，避免填满
3. 观察元素分布，规划消除顺序
4. 合理使用洗牌和提示道具
5. 注意时间限制，不要在单个选择上犹豫太久
